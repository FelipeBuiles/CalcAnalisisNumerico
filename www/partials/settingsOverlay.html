In this section you will find the way of using every operation, function or constant avaliable on our application.

Operations:
Seno = sin(x), sen(x)
Arc Seno = asin(x), sen(x)
Coseno = cos(x)
Arc Coseno = acos(x)
Tangente = tan(x)
Arc Tangente = atan(x)
Raiz = sqrt(x)
Logaritmo = log(x)
Valor absoluto = abs(x)
Exponente = ^
Concat = ||
Module = %

Functions:
Random = random(x)
Factorial = fac(x)
Minimum = min(x)
Maximum = max(x)
Pow = pow(x,y)

Constants:
Epsilon = e, E
Pi = pi, PI
